This is a nonogram solver with a web UI.

## Features:
- Selectable grid size
- Error handling - unsolvable nonograms
- Nonogram solving
- Changeable parameters
- Responsive(ish)
- Deployed to [GitHub pages](https://0l1v3rr.github.io/nonogram-solver/) - available 24/7
